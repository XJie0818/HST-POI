**Mobile Phone**：The Mobile Phone dataset tracks the movements of app users and corresponding timestamps.

&#8226; You can get the raw dataset from: https://zenodo.org/records/10836269

**Gowalla**：The Gowalla dataset included user check-in and friend relationship information. The data were collected from February 2009 to October 2010, which contained 6,442,890 check-in records.

&#8226; You can get the raw dataset from: https://snap.stanford.edu/data/loc-gowalla.html

**Traffic Camera**：The Traffic Camera dataset contains the vehicle’s mobility records between March 2021 and April 2021, detailing people’s daily trips.

&#8226; The Traffic Camera dataset used in this study is derived from the work presented in [1].
